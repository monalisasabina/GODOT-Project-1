# My First Godot Project

A simple 2D game project built with Godot 4.6 to learn game development fundamentals.

## About

This is a learning project to get familiar with:
- Godot Engine basics
- GDScript programming language
- Game development workflow
- Version control with Git

## Tech Stack

- **Engine**: Godot 4.6
- **Language**: GDScript
- **Version Control**: Git + GitHub
- **Code Editor**: VS Code / Godot built-in editor

## Setup

1. Install [Godot 4.6](https://godotengine.org/download)
2. Clone this repository:
```bash
   git clone <your-repo-url>
```
3. Open Godot and import the project
4. Click "Import" and select the `project.godot` file

## Running the Project

1. Open the project in Godot
2. Press F5 or click the ▶ play button
3. The game window will launch

## Project Structure
```
MyFirstGame/
├── scenes/          # Game scenes (.tscn files)
├── scripts/         # GDScript files (.gd)
├── assets/          # Images, sounds, etc.
└── project.godot    # Godot project file
```
